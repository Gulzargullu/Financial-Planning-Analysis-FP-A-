# Financial-Planning-Analysis

This project analyzes financial performance using Budget vs Actual data to identify variances, risks, and department-level insights.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Power BI
- Excel

## 📁 Project Structure
- Data generation (Python)
- Data cleaning & transformation
- Variance analysis
- Dashboard creation in Power BI

## 📈 Dashboard Features
- Executive Summary KPIs
- Department Performance Analysis
- Risk & Trend Monitoring

## 🔍 Key Insights
- IT and Sales show highest variance
- 25% data classified as high risk
- Overall variance: -1.58%

## 📌 Outcome
Built a full FP&A reporting system simulating real corporate financial analysis.
